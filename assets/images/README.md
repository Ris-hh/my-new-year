# 图片资源文件夹

请将需要显示的图片文件放置在此文件夹中。

## 使用方法

1. 将图片文件命名为：`photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`, `photo5.jpg`
2. 支持的图片格式：`.jpg`, `.jpeg`, `.png`, `.webp` 等浏览器支持的格式
3. 图片会自动加载并显示在 3D 场景中

## 修改图片数量

如需修改加载的图片数量和文件名，请编辑 `index(1).html` 文件中的 `loadFixedPhotos()` 函数，修改 `imageFiles` 数组。

## 注意事项

- 图片文件名区分大小写
- 建议图片尺寸不要过大（推荐 800x600 到 1920x1080 之间）
- 如果图片加载失败，会在浏览器控制台显示警告信息
