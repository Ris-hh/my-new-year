# 音频资源文件夹

请将背景音乐文件放置在此文件夹中。

## 文件要求

1. **music1.mp3** - 第一首背景音乐（MP3格式）
   - 或者 **music1.ogg** - 第一首背景音乐（OGG格式，用于更好的浏览器兼容性）

2. **music2.mp3** - 第二首背景音乐（MP3格式）
   - 或者 **music2.ogg** - 第二首背景音乐（OGG格式，用于更好的浏览器兼容性）

## 播放方式

- 两首音乐会顺序循环播放
- 第一首播放完毕后自动播放第二首
- 第二首播放完毕后自动回到第一首
- 如此循环

## 支持的格式

- MP3 (推荐)
- OGG (用于更好的浏览器兼容性)

## 注意事项

- 如果文件不存在，音乐功能将无法使用
- 建议使用较小的文件大小以优化加载速度
- 确保音频文件格式正确，浏览器可以正常播放
